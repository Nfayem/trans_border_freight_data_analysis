**Kolmogorov-Smirnov (KS) test**

The KS test checks whether two samples come from the same distribution.  
- **Null Hypothesis (H‚ÇÄ):** The sampled data follows the same distribution as the full dataset.  
- **Alternative Hypothesis (H‚ÇÅ):** The sampled data is significantly different from the full dataset.  

If **p-value > 0.05**, we **fail to reject H‚ÇÄ** ‚Üí The sample preserves the original distribution.  
If **p-value < 0.05**, we **reject H‚ÇÄ** ‚Üí The sample does not preserve the original distribution.


# Define sampling percentages
sample_sizes = [0.0001, 0.0002, 0.0003, 0.0005, 0.05, 0.1, 0.2]
features = ["SHIPWT"]

# Store KS test results
ks_results = {}
